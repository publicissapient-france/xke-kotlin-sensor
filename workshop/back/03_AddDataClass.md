# Add a Data Class

We want to manage data from smartphone sensors. The first step is to create a [Data Class](https://kotlinlang.org/docs/reference/data-classes.html).

We will need the following values in our Data class:

| Name  | Type |
| ------------- | ------------- |
| smartphoneId  | UUID  |
| type  | String  |
| eventTime  | Instant  |
| value  | String  |
